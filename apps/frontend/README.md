# Frontend Plonio 

Sistema de gestión de proyectos

## Descripción

El sistema de gestión de proyectos es una herramienta que permite a los usuarios gestionar proyectos de manera eficiente y organizada.

## Funcionalidades

- Creación de proyectos
- Gestión de tareas
- Gestión de usuarios
- Gestión de roles
- Gestión de permisos

## Tecnologías utilizadas

- React
- Vite
- TypeScript
- TailwindCSS
- Storybook
- Vitest

## Arquitectura

- Clean Architecture
- Visual Test Driven Development
